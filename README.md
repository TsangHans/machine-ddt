# machine-ddt

> 用机器的角度看弹弹堂？（仅供学习和交流，切勿用于商业用途）。
> 有想法一起开发请联系作者

## 计划开发的接口（如有合适的，可新增）

- [ ] 判断当前游戏画面的状态（主页面，游戏大厅，房间，副本房间第几关，加载页面，游戏中玩家回合，游戏中非玩家回合）
- [ ] 识别风力、角度（提高识别精确度）
- [ ] 计算发射力度（提高拟合精确度）
- [ ] 目标距离检测（自动读取检测、手动指定检测）
- [ ] 发射（自适应环境因素，提高发射准确率）
- [ ] 识别当前任务列表
- [ ] 开启房间（竞技、副本）
- [ ] 邀请好友（房间内）
- [ ] 识别施邀对象，接收或拒接邀请
- [ ] 指定任务名称，完成任务或接取任务
- [ ] 经典服自动验证挂机检测
